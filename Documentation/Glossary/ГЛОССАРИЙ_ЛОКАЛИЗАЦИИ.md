# Глоссарий локализации

> Термины пайплайна локализации (русская локаль UT2004). Термины игровых сущностей — в глоссариях по entity (см. [_СПИСОК_ГЛОССАРИЕВ](../_СПИСОК_ГЛОССАРИЕВ.md)).

## Термины

| Термин | Определение |
|--------|-------------|
| **Базовый файл (Base File)** | Исходный файл, с которого переводится .rut: .int из `/int` или .est из `/est` с тем же базовым именем (например `Foo.int` или `Foo.est` → `Foo.rut`). |
| **Валидация** | Проверка перевода по глоссариям сущностей и по [File Translation Status Rules](../File%20Translation%20Status%20Rules.md) (формат, статус). |
| **Глоссарий сущности** | Документ в [Documentation/Glossary/](.) с терминами одной группы (Game_*, UI_*, Lore_*, Tech_*). |
| **Интеграция** | Копирование готовых .rut в установку игры. |
| **Исходный язык** | Язык строк в `/int` (английский). Для файлов без .int источником может быть `/est`. |
| **Ключ (key)** | Имя поля в .int/.est/.rut (например `GameName=`, `Caption=`). Переводимое значение — строка в кавычках справа от `=`. |
| **Локализация (файл)** | Файл .rut в `/rut` — русский перевод одного исходного .int или .est. |
| **Old Russian (колонка)** | В таблицах глоссариев сущностей — строка из старой русской локализации (/rut_old). Только для визуального сравнения; **никогда не использовать как источник текста для перевода** в .rut. |
| **/rut_old** | Папка со старой русской локализацией. Можно обращаться к ней для сравнения; **не источник перевода** — строки оттуда не переносим в .rut. |
| **Статус (файла)** | Класс по [File Translation Status Rules](../File%20Translation%20Status%20Rules.md): No .rut, No .int, Untranslated, Spanish!!!, No Caption, In Progress, Done. |
| **Сущность (entity)** | Группа терминов в глоссарии (карты, оружие, меню и т.д.). Список — [_СПИСОК_ГЛОССАРИЕВ](../_СПИСОК_ГЛОССАРИЕВ.md). |
| **Целевой язык** | Русский (ru); выходная локаль — папка `/rut`. |
| **`; EN:`** | Строка-комментарий в .rut перед переводом, хранящая эталонную английскую строку (из .int/.est). Обязательна для переводимых ключей. |
