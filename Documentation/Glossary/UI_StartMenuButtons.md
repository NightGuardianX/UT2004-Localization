# UI Start menu buttons glossary

## Entity Glossary Rules (UI_StartMenuButtons)

- **Sources:** `XInterface.int` (`[UT2MainMenu]`), `GUI2K4.int` (`[UT2K4MainMenu]`). Current Russian: primary source `GUI2K4.rut`; Old Russian: `XInterface.rut` in `/rut_old/`.
- **Scope:** Primary buttons on the main screen: Host, Instant Action, Multiplayer, Quit, Settings, Single Player. Main screen itself → `UI_Menus` as `UT2k4MainMenu`.
- **Exclusions:** Submenu and mid-game menu entries → other UI entities.
- **Table:** 5 columns.

Table format and filling: [GLOSSARY_RULES — Glossary row format](../Glossary_Rules.md#glossary-row-format).
