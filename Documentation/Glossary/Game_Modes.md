# Game modes glossary

## Entity Glossary Rules (Game_Modes)

- **Sources:** XGame.int, BonusPack.int, SkaarjPack.int, UT2k4Assault.int, OnslaughtFull.int.
- **Categories:** Base gamemodes (Deathmatch, Team DM, CTF, Domination, Bombing Run, Onslaught, Assault); Other (variants).
- **Scope:** Base game types in standard Game Type selection only. Mutators and ladder variants excluded.
- **Table:** 5 columns. Each distinct GameType class gets its own row.

Table format and filling: [GLOSSARY_RULES — Glossary row format](../Glossary_Rules.md#glossary-row-format).
