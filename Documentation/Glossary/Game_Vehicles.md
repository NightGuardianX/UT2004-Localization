# Vehicles glossary

## Entity Glossary Rules (Game_Vehicles)

- **Sources:** OnslaughtFull.int, Onslaught.int, Vehicles.int.
- **Categories:** By source (Onslaught base, OnslaughtFull, OnslaughtBP). Only vehicle display names.
- **Scope:** Display names in UI (scoreboard, HUD, vehicle selection) only. Helper messages (e.g. “Press [Use] To Enter Vehicle.”).
- **Table:** 5 columns.
