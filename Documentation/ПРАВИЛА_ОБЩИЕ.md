# Общие правила документации

> Пайплайн локализации UT2004 (русская локаль). Эталон принципов — [__новый_проект/_общее/ОБЩ_ПРИНЦИПЫ_ДОКУМЕНТАЦИИ.md](../__новый_проект/_общее/ОБЩ_ПРИНЦИПЫ_ДОКУМЕНТАЦИИ.md).

## Принципы

1. **Карточка описывает своё** — документ содержит суть одного элемента; списки и индексы — в `_СПИСОК_*`.
2. **Ссылка вместо повтора** — информация в одном месте, остальные ссылаются.
3. **Связей — не больше пяти** в разделе «Ключевые связи»; полный перечень — в индексах.
4. **Шаблоны обязательны** — новые документы только по `_ШАБЛОН_*.md` (см. [Templates/](Templates/)).
5. **Префикс `_`** — служебные файлы и папки (индексы, шаблоны).
6. **Имена без пробелов** — только подчёркивания.

## Глоссарии

- Определения терминов — **только в глоссариях**; в остальных документах — ссылки.
- Индекс глоссариев: [_СПИСОК_ГЛОССАРИЕВ.md](_СПИСОК_ГЛОССАРИЕВ.md).
- При новом термине — добавить в соответствующий глоссарий.
