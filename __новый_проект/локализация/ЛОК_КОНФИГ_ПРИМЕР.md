# Пример конфига: Пайплайн локализации

> Заполнить под целевой проект. Используется промптом `ЛОК_PROMPT_ИНИЦИАЛИЗАЦИЯ.md`.

## Вопросы при инициализации

Ответить перед запуском промпта (или оставить пустым — промпт запросит):

| Вопрос | Ответ | Пример |
|--------|-------|--------|
| Интеграция с движком | Встроенная / внешняя / гибрид | Внешняя, текстовые файлы |
| Инструменты | Crowdin, Lokalise, POedit, скрипты, AI | POedit, AI-агент для итераций |
| Формат файлов | PO, CSV, JSON, кастом (напр. UE2) | PO (по умолчанию) |
