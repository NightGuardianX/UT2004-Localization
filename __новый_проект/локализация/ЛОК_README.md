# Пайплайн локализации

> Отдельный репозиторий. Локали и документация для них — отдельные категории документов.

Общие принципы: `_общее/ОБЩ_ПРИНЦИПЫ_ДОКУМЕНТАЦИИ.md`.

## Файлы

| Файл | Назначение |
|------|------------|
| [ЛОК_PROMPT_ИНИЦИАЛИЗАЦИЯ.md](ЛОК_PROMPT_ИНИЦИАЛИЗАЦИЯ.md) | AI-промпт инициализации; содержит вопросы про движок, инструменты, форматы |
| [ЛОК_ЧЕКЛИСТ.md](ЛОК_ЧЕКЛИСТ.md) | Пошаговый план: анализ → глоссарий → валидация → перевод → ревью → интеграция |
| [ЛОК_КОНФИГ_ПРИМЕР.md](ЛОК_КОНФИГ_ПРИМЕР.md) | Пример конфига: языки, неймспейсы/сущности, форматы, инструменты |

## Процесс (кратко)

1. Анализ существующего — что есть
2. Обновление и коррекция глоссария
3. Валидация всего через глоссарий
4. Перевод (ручной / итерация через AI-агента)
5. Итерация валидации перевода и глоссария
6. Ревью
7. Интеграция
